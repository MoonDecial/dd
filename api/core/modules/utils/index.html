useTime​

▸ useTime(): number

Get the real time since the start of the animation.

Remarks

The returned value accounts for offsets caused by functions such as waitFor.

Example

// current time: 0s
yield* waitFor(0.02);

// current time: 0.016(6)s
// real time: 0.02s
const realTime = useTime();

Returns​

number
